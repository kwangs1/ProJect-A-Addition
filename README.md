# 개발환경

● Server OS : Windows10

● Language : JAVA 1.8

● WEB Server : Apache

● WAS Server : Tomcat 9.0.11

● build tool : maven 2.5.1

● DB: Oralce XE 11g-winx64

● FrameWork : Spring 4.1.1.RELEASE

● ORM : mybatis 3.1.0

● 회원가입

  ⤷ 이메일 인증 구현
  
  ⤷ 유효성 검사

● 게시판 댓글

  ⤷ Ajax를 이용한 댓글 및 답글 CRUD

  ⤷ 별점 기능
